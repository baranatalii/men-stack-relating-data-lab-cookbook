# FRAGRANCES

## A Fragrance web application for perfume & cologne enthusiasts to manage thier personal collections & explore scents from the community.

## Features

- `Personal Vault :` Add. edit & delete fragrances in your private collection.
- `Community Access:` Explore other users' collections to find new scents.
- `User Authentication:` Secure sign-up & login functionality.

## Tech

- **Node.js Express.js**
- **MongoDB** [ With Mongoose ]
- **EJS** [ Embedded JavaScript ]
- **Styles**: CSS Variables & Gradients
- **Auth: Express & Bcrypt**

## Installation

- 1. Clone the repository.
- 2. Run npm install to install dependencies
- 3. Create a .env file and add your MONGODB_URI and SESSION_SECRET
- 4. Run npm start or nodemon to launch the app
- 5. Open http://localhost:3000 in your browser.
